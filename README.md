## üöÄ Getting Started

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/NISCHAY17/hackclubportfolio.git
   cd hackclubportfolio
   ```

2. Run a local server:
   ```bash
   cd portfolio
   python3 -m http.server 8000
   ```

3. Open in your browser:
   ```
   http://127.0.0.1:8000/
   ```

## üì¶ Deployment on GitHub Pages

### Setup Instructions

1. **Ensure `index.html` exists** in the `portfolio/` folder:
   ```bash
   mv portfolio/main.html portfolio/index.html
   ```

2. **Commit and push your code:**
   ```bash
   git add .
   git commit -m "Deploy portfolio to GitHub Pages"
   git push origin main
   ```

3. **Configure GitHub Pages:**
   - Go to your repository on [GitHub](https://github.com)
   - Click **Settings** ‚Üí **Pages**
   - Under **Source**, select `main` branch
   - Set the folder to `/portfolio`
   - Click **Save**

4. **Wait for deployment** (usually 1-2 minutes)

5. **Your site will be live at:**
   ```
   https://<your-username>.github.io/<repo-name>/portfolio/
   ```
